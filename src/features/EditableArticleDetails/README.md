## EditableArticleDetails feature

This feature provides the functionality to view and edit article content.

### Public api

#### Components

- `EditableArticleDetails` - A component that provides an interface for viewing and editing the content of an article.

- `ArticleAdditionalInfoContainer` - A container component that displays additional article information.

#### Slice

- `articleReducer` - The reducer responsible for managing the state of the editable article.

#### Types

- `ArticleRootSchema` - A type defining the structure of the article slice in the store.
