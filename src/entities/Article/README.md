## Article entity

This module handles the representation and management of articles. It provides components for rendering detailed article information, displaying lists with sorting and filtering options, controlling the view mode.

### Public api

#### Components

- `ArticleDetails` - A component displaying detailed information about the article.

- `ArticleList` - A component for displaying a list of articles.

- `ArticleViewSelector` - A component for switching the article list display mode (grid, list).

- `ArticleSortSelector` - A component for selecting the sorting criteria for the list of articles.

- `ArticleTypeTabs` - A component for selecting the article type/category via tabs.

- `ArticleAdditionalInfo` - A component for displaying supplementary data about the article.

- `ArticleDetailsHeader` - A header component for the article details page containing auxiliary buttons.

#### Types

- `Article` - A type describing the article object.

- `ArticleView` - A type describing the display mode of the article card.

- `ArticleType` - A type describing the content-based categories of articles.

- `ArticleSortField` - A type describing the fields available for sorting articles.
